# Contributing

Thanks for considering contributing to this repository! To make contributions easy to review and merge, please follow these steps.

1. Fork the repository.
2. Create a topic branch (git checkout -b feature/my-change).
3. Make changes and include tests where applicable.
4. Run the test suite locally (pytest).
5. Push your branch and open a pull request with a clear description of your changes.

For small edits (typos, docs), you can also propose changes directly via GitHub's web editor.
